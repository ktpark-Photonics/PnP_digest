# AGENTS.md

## 언어 규칙
- 사용자와의 모든 대화, 계획, 설명, 진행 보고는 항상 한국어로 작성한다.
- 특별히 요청받지 않는 한 영어로 답하지 않는다.
- 코드 주석과 README도 기본적으로 한국어로 작성하되, 코드 식별자와 라이브러리 이름은 원문을 유지한다.

## 작업 원칙
- 먼저 계획을 제시한 뒤 구현한다.
- 한 번에 전체를 만들지 말고 단계별로 구현한다.
- Python 중심으로 구현한다.
- 타입 힌트와 docstring을 포함한다.
- 테스트 가능한 코드는 tests를 추가한다.
- 특허 사실을 지어내지 않는다.
- 특허번호, 제목, 출원인, 날짜, 요약 일치 여부를 검증 가능한 구조로 설계한다.
- 논문 figure는 항상 재사용 가능하다고 가정하지 않는다.
- 구조화 JSON 스키마를 먼저 만들고, 문서 생성은 그 다음에 한다.
