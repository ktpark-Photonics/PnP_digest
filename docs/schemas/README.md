# Schemas

이 디렉터리는 `pnp-digest export-schemas` 명령으로 생성된 JSON schema snapshot을 저장하는 위치다.

Phase 0에서는 명령을 통해 아래 모델 스키마를 내보내도록 설계했다.

- `PipelineRun`
- `RawSourceRecord`
- `DocumentRecord`
- `RelevanceAssessment`
- `VerificationResult`
- `SummaryPayload`
- `FigureAsset`
- `ReviewTask`
- `OutputBundle`
- `IngestArtifact`
- `NormalizedArtifact`
